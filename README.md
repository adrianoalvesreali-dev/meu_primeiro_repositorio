
# Meu primeiro repositório

Este repositório foi criado para testar a integração entre GitHub e OpenClaw.

## Visão geral

A ideia deste projeto é validar, na prática, como o OpenClaw pode interagir com o GitHub para ler arquivos, consultar commits, analisar repositórios e ajudar em tarefas de automação.

## Objetivo

Testar a comunicação entre GitHub e OpenClaw de forma simples, didática e funcional.

## O que já foi testado

- criação do repositório
- criação do arquivo README.md
- leitura de arquivos pelo OpenClaw
- resumo de conteúdo do README
- listagem de commits do repositório
- integração inicial com a skill do GitHub

## Como usar

No OpenClaw, você pode fazer pedidos como:

- listar arquivos do repositório
- abrir o README.md
- resumir o conteúdo do projeto
- mostrar commits
- consultar detalhes do repositório

## Exemplo de uso

Exemplos de comandos no OpenClaw:

- mostre os arquivos do repositório meu_primeiro_repositorio
- abra o README.md do repositório meu_primeiro_repositorio
- mostre os commits do repositório meu_primeiro_repositorio
- resuma o conteúdo do README.md do repositório meu_primeiro_repositorio
